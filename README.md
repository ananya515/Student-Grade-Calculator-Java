# 🎓 Student Grade Calculator (Java)

## 📌 Project Overview
This is a simple Java console-based application that calculates:
- Total Marks
- Average Percentage
- Grade based on the average score

## 🚀 Features
✔ Accepts marks input for multiple subjects  
✔ Validates input (marks must be between 0–100)  
✔ Calculates total marks  
✔ Computes average percentage  
✔ Assigns grade (A–F) based on performance  

## 🛠 Technologies Used
- Java
- Scanner Class
- Loops (for loop)
- Conditional Statements (if-else)

## 📊 Grading Criteria
- 90% and above → A  
- 80% – 89% → B  
- 70% – 79% → C  
- 60% – 69% → D  
- 50% – 59% → E  
- Below 50% → F
